Getting Started

There are 2 components that are needed in order to be able to play Classic Doom.

A Source Port

A source port is a fan-made modification of the original Doom engine which was open-sourced in 1997. These are available for free, and you can grab a copy of the Crispy Doom source port in the Download page.

A WAD file

While the engine was made available for free, the assets that make up the game (textures, music, maps etc...) were not. You still need an original copy of Doom in order to play the original game. The assets are contained in a WAD file. Crispy Doom can play nearly all variants of Doom. If you don't own any, you may download the Shareware version of Doom, extract it and copy the DOOM1.WAD file into your Crispy Doom directory. Alternatively, you may want to play Crispy Doom with Freedoom and a MegaWAD.

New Cheat Codes

In addition to the default DOOM cheat codes, Crispy Doom offers additional ones that offer extra variation to the gameplay. Keep in mind that cheat codes are disabled on the "Nightmare!" skill level, except for the iddt and idclev cheats.

Compatiblity

Crispy Doom strives for maximum compatibility with all limit-removing vanilla maps, but does not support Boom or ZDoom maps.
