  • Get the flatpak from flathub.org
  • Install using apt on Ubuntu/Debian based systems: apt-get install crispy-doom

MacOS

  • To install on OSX, get macports and then run: sudo port install crispy-doom

Build from source

Build instructions for Debian-based systems

First download the source code, then get the pre-requisites:

  • sudo apt install build-essential automake git
  • sudo apt build-dep crispy-doom

Then start the compilation:

  • cd crispy-doom
  • autoreconf -fiv
  • ./configure
  • make

Building on Windows
